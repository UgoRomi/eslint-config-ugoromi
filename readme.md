# To Install:

set:
<code>"eslintConfig": {
"extends": [
"ugoromi"
]},</code> in your package.json  
then run
`npx install-peerdeps --dev eslint-config-ugoromi`

## For Typescript:

Just need to extend `ugoromi/typescript` instead of `ugoromi`
